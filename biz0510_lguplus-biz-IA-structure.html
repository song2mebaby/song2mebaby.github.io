03 페이지 유형별 분류
3-1

홈(메인)

/biz
브랜드 인지, 솔루션 노출, 리드 유입 허브
  • Hero 슬라이딩 상품 카드 + 인사이트 그리드
  • 업종/유형별 솔루션 탐색기
  • 고객 도입 성과 수치 섹션
  • 인사이트 카테고리 퀵 링크
  • Floating 퀵메뉴 CTA
3-2

상품 목록

/biz/all/telecom/{카테고리}
카테고리별 상품 탐색
  • GNB 전체메뉴 활용
  • 상품 수 55개 이상
3-3

상품 상세

/biz/all/{분류}/{상품명}/{코드}
개별 상품 상세 정보 및 상담 전환
  • 좌측 앵커 메뉴 (상품개요/특장점/요금/FAQ)
  • Breadcrumb 네비게이션
  • 간편 상담 신청 CTA
3-4

솔루션 (업종별)

/biz/solution/industry/{업종}
업종 니즈 기반 솔루션 세트 제안
  • retail / manufacturing / healthcare
  • platform / public-sector / finance / education
3-5

솔루션 (유형별)

/biz/solution/type/{유형}
기술/기능 유형 기반 솔루션 소개
  • aidc / aicc / security / office
3-6

인사이트 목록

/biz/insight/{type}
콘텐츠 마케팅 허브
  • story / trend / success-case / news / webinar / event
  • 카테고리 탭 + 3열 카드 그리드 + 페이지네이션
3-7

인사이트 상세

/biz/insight/{type}/{id}
콘텐츠 상세 및 관련 콘텐츠 연결
  • 상단 태그·등록일 + 본문
  • 관련 콘텐츠 추천
3-8

고객지원 허브

/biz/support
모든 지원 채널 한 페이지 안내
  • 고객센터 번호 배너
  • 상담신청 / 1:1문의 / FAQ 카드
  • 셀프서비스 가이드 + 장애 지원
3-9

MY 페이지

/biz/mypage
요금 조회, 상품 관리, 법인폰/총량 관리
  • 로그인 필요 (U+ ID)

06 모바일(m.lguplus.com/biz) 구조 특이사항
GNB: 하단 탭 바 또는 햄버거 메뉴로 전환
Hero 섹션: 단일 카드 슬라이드(가로 스크롤)
섹션 카드: 가로 스크롤 Carousel로 전환

상품 상세 앵커 메뉴: 좌측 사이드바 → 상단 수평 탭으로 전환
07 IA 분석 기반 현황 이슈 체크리스트
  • 발견성 상품 카테고리 깊이: GNB 전체메뉴 기준 상품 수 55개 이상 — 사용자가 원하는 상품을 발견하기 어려울 수 있음
  • 개념 혼재 '솔루션' 메뉴와 '상품' 메뉴의 개념 경계가 사용자 관점에서 불명확
  • 복잡도 MY 메뉴 하위 메뉴 10개 카테고리 이상 — B2B 특성상 다양하나 우선순위 정리 필요
  • 일관성 검색 진입점: GNB 검색 아이콘 + 메인 검색 블록 두 군데 존재 (일관성 검토 필요)
  • UX 개선 인사이트 카테고리 6개 구분 — 통합 허브 UX 개선 가능
  • 모바일 PC 기준 대분류 5개에 대한 모바일 최적화 검토 필요
