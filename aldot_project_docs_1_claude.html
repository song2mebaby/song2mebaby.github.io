요구사항 정의서 (PRD)

알닷(uplusmvno.com) 모바일웹 UX/IA 개편 프로젝트의 요구사항을 정의합니다. 현황 역설계 분석 결과를 기반으로 작성되었습니다.

1. 프로젝트 개요 및 배경
프로젝트명
알닷(알뜰폰닷컴) 모바일웹 UX/IA 개편
대상 서비스
uplusmvno.com (알닷, SPA · React 계열)
개편 배경
  • 하단 탭바 5개 ↔ 햄버거 메뉴 7개 간 중복·불일치로 사용자 혼란
  • 핵심 전환 액션(유심구매)이 하단탭에 미노출
  • 요금제 필터 과부하 → 첫 이용자 진입 장벽
  • 가입 플로우 13단계 → 중간 이탈 위험
  • 알닷케어·마이페이지 역할 중복으로 구조적 혼란
프로젝트 목표
  • 네비게이션 구조 단순화 (7 depth→5 depth)
  • 핵심 전환 경로 (유심구매) 접근성 강화
  • 요금제 탐색 UX 개선 → 필터 단순화
  • 가입 플로우 13→8단계 압축
  • 알닷케어 + 마이페이지 기능 통합
