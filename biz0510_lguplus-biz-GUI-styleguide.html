07 각 화면별 GUI 상세
7-1 홈(메인) 화면 섹션 구성
Section 1: Hero Grid
좌(2/3): 상품 슬라이드 카드 — Purple 배경, 상품명 대형
우상(1/3): 인사이트 카드 — 연보라/흰배경
우하좌: 보도자료 카드 — 다크
우하우: 검색 블록 — 다크 배경, 보라 버튼
Section 2: 업종/유형 솔루션
헤딩: 60px Bold
필터 탭: 업종(활성)/유형(비활성) — 다크 pill 버튼
업종 카드: 선택 보라, 미선택 회색
hover 시 Purple 활성화
Section 3: 도입사례
인용 블록: 수치 강조 (색상/Bold)
좌우 화살표 슬라이드
회사명 + 수치 + 설명
Section 4: 인사이트 퀵링크
6개 항목 수평 나열
각 항목에 "최신글 보기" 링크
블로그/뉴스레터/보도자료/웨비나/고객사례/이벤트
7-2 상품 상세 화면 구성
Header 영역
Breadcrumb — 회색 소형 텍스트
상품명 대형 헤딩 + 마침표 애니메이션
상품 설명 서브텍스트
우측: 북마크/링크복사/장바구니 + CTA 보라 버튼
좌측 앵커 메뉴 (PC)
세로 사이드바 형태
현재 섹션: 보라 텍스트 or 왼쪽 border 강조
항목: 상품개요/특장점/이용요금/추천고객/주요기능/FAQ
본문 영역
